# Game: Whac a Mole
In this tutorial, you will learn to create the whac a mole game with html, css, and javascript. Specifically, you will learn how to code the game using html5 canvas.
While creating this mario themed whac-a-mole game, you will learn how to use javascript to set the game board by editing dom elements, add click handlers to whack the mole, and randomly spawn the mole and piranha plant.

Technoly used:

HTML:

• Builds the structure of the game

• Displays the title, score counter, restart button, and the 3×3 game board

CSS:

• Styles the game interface

• Designs the board, tiles, background, and button

• Controls layout, size, and appearance of images

JavaScript:

• Handles the game logic

• Randomly places the mole and piranha plant

• Updates the score

• Detects game over conditions

• Controls the restart functionality

Web Browser:

• Runs the game (Google Chrome, Microsoft Edge, Mozilla Firefox)

• No frameworks or external libraries are required


Instructions to run the game:

Create a project folder:

• Place all required files in one folder:

• index.html

• mole.css

• mole.js

• Image files (mario-bg.jpg, soil.png, pipe.png, monty-mole.png, piranha-plant.png)

Launch the game:

• Double-click index.html, or Open it using any modern web browser

Play the game:

• The game starts automatically

• Click the mole to gain 10 points

• Avoid clicking the piranha plant

Game Over:

• Clicking the piranha plant ends the game

• The final score is displayed on the screen

Restart:

• Click the Restart Game button to reset the score and play again
